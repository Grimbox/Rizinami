Aide à l'utilisation

Le but de cette application est de 'traduire' une liste de numéros INAMI dans une structure particulière, définie dans les documents de spécifications, dans le cadre des accréditations. L'application originale développée par l'INAMI permet plusieurs choses:

  • Rechercher des activités dans un intervalle de temps donné;
  • Rechercher une activité dont les données ont été précédemment envoyées;
  • Envoyer un ensemble de données correspondant à une activité.

Ce que l'application fait

L'application courante se charge de mettre les données en forme pour qu'elles soient acceptées par l'application INAMI. Les formats pris en charge sont décrits à la page 14 du document de spécifications. Elle génère donc un document texte proprement formaté pour un contexte particulier. Voir les contraintes ci-dessous :)

Ce que l'application ne fait pas encore

(Non, ce n'est pas encore parfait):

  • Elle ne gère pas les "non participants"
  • Elle n'exporte pas les informations dans le format XML
  • Elle ne gère pas le rôle du participant; un rôle par défaut ("participant") est donné à chaque membre. Ce comportement n'est valable que dans le cadre de l'accréditation des dentistes, ce rôle étant le seul admis.
  • Elle ne vérifie pas la validité des numéros INAMI et génèrera donc un fichier avec les numéros que l'utilisateur lui aura donné.
  • Seuls les numéros INAMI sont acceptés comme identifiant d'un participant. Les numéros de registre national ne sont pas gérés.
